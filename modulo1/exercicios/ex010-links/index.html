Usando links externos

Você pode acessar o meu repositório público no GitHub

Você também pode acessar o meu canal gratuito no YouTube

Ultilizamos a tag <a> para criar links e coloque os parametros target="_blank" rel="external" para abrir em uma nova aba.

Usando links interno

Essa é a primeira página do meu site. Se você quiser pode acessar também a minha segunda página.

Basta criar um novo arquivo na mesma pasta e fazer um link pra ele e coloque o rel="next" na primeira página e o rel="prev" na segunda página.


Você também pode acessar a nossa página de notícia.

Essa página fio criada dentro de uma pasta. Você pode acessar ela normalmente com link usando o ctrl + space para escolher, porém ao criar o link voltar na outra página tem que colocar o ../ para achar o arquivo.
