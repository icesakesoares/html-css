Principais formatações

Negrito

Nessa frase temos um termo em negrito usando a tag <b> (não semantica).

Nesta frase temos um termo em destaque usando a tag <strong> (semantica).

Itálico

Nesta frase temos um termo em itático usando a tag <i> (não semantica).

Nesta frase temos um termo em enfase usando a tag <em> (semantica).

Texto marcado

Podemos também criar um texto marcado usando a tag <mark> e com estilo CSS dentro da tag.

E no outro parágrafo temos outro texto marcado sem o estilo CSS.

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido mais não considerado usando a tag <del>.

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado ao texto depois usando a tag <ins>.

Existe também o sublinhado com a tag <u> (não semantica).

Texto sobrescrito

Para inserir coisas do tipo x20+3 usando a tag <sup>.

Texto subescrito

Para inserir coisas do tipo H2O usando a tag <sub>.
